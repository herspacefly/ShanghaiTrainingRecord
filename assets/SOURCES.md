# 媒体资源来源与替换说明

## 当前版本

当前运行环境无法连接 Wikimedia Commons 和 AutoGLM 图片检索服务，因此没有使用无法核验许可的远程照片或音乐。

- `images/*.png`：Codex 在本地生成的地点档案示意视觉，不代表地点实景，无外部版权依赖。
- `audio/*-placeholder.wav`：本地生成的 1 秒有效静音 WAV，用于验证播放、暂停、循环和地点切换逻辑。

## 正式提交前 TODO

1. 将每张示意图替换为对应地点的实训照片，保留现有文件名即可。
2. 将静音 WAV 替换为拥有明确使用许可的器乐或环境音，并同步修改 `data/poi-data.js` 中的文件名与来源。
3. 对每项外部素材记录：作品标题、作者、原始链接、许可名称、下载日期。
4. 不要直接热链搜索结果，也不要使用无法确认授权范围的图片或音乐。

## 建议检索来源

- Wikimedia Commons：https://commons.wikimedia.org/
- 各参访单位官方网站的媒体资料页
- 用户本人或实训小组拍摄的照片与录音
