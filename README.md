# 上海实训地图

## 正确打开方式

不要直接双击 `index.html`。本项目使用 ES Modules，`file://` 协议会阻止模块加载。

Windows 下双击项目目录里的 `start-local-server.bat`，脚本会启动本地静态服务器并打开：

`http://127.0.0.1:4173/`

也可以在 `Project` 目录执行：

```powershell
python -m http.server 4173 --directory .
```

然后访问上面的地址。若使用 Nginx，将 `Project` 配置为站点根目录后，通过 Nginx 的端口访问。

## 当前实现成果

本项目已经完成一个可静态部署的“上海实训沉浸式地图”首版，用于记录四天上海实训中的历史教育、智慧港口、高新企业和科研院所参访路线。

### 已实现功能

- 百度地图 GL 全屏地图场景，支持缩放、地图控件和地点标记。
- 共 9 个实训地点，按 D1-D4 四天组织，并区分重点参访地点。
- 点击地图标记后聚焦地点、打开地图信息窗，并切换右侧地点故事。
- 地点详情包含参访时间、地点类型、摘要、观察重点、信息标签、官网链接和媒体来源说明。
- “全部 / D1 / D2 / D3 / D4”行程筛选，会同步更新地图标记、路线、时间线和图表高亮。
- 路线回放支持播放、暂停、继续和重播，显示移动端点并渐进绘制路线。
- 点击路线分段可查看交通方式、估算距离和耗时。
- 侧边栏提供“概览 / 地点 / 洞察”三个视图，桌面端可收起，移动端为底部面板。
- 使用 ECharts 实现每日里程柱状图、地点类型分布环形图和每日参访时间图。
- 支持按地点切换音频、循环播放、静音/暂停及浏览器播放失败提示。
- 对地图、图表、图片、音频和 `file://` 直接打开方式提供错误或降级提示。
- 已适配桌面端和移动端布局，并提供键盘焦点样式、按钮标签和减少动画模式。

### 当前数据与素材状态

- 坐标、时间、里程和交通方式是首版模拟/估算数据，代码中保留了 `TODO` 标记。
- 地点图片是本地生成的示意视觉，并明确标注为“待替换实训照片”。
- 音频是可播放的静音占位 WAV，用于验证播放、暂停、循环和地点切换逻辑。
- 正式提交前可直接替换 `assets/images/` 和 `assets/audio/` 中的文件，并同步更新 `assets/SOURCES.md`。
